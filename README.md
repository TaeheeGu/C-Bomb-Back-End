# C-Bomb, 폭발하는 채팅창

타인과 단발성 교류를 위한 채팅 서비스

필요에 따른 타인과 일시적인 교류를 통해 지속적인 인간관계에서 오는 스트레스를 최소화 합니다.

누군가와 영화를 보고싶을 때,

하지만 주변에 아무도 없을 때,

우리는 그냥 오늘 같이 영화만 볼 사람이 필요합니다.

### 휘발성

우리는 지속적인 관계가 부담스럽습니다.

### 활동성

하지만 우리는 같이 영화 볼 사람이 필요합니다.

### 익명성

그러나 나의 정보를 노출하고 싶지 않습니다.

### 신뢰성

믿을 수 있는 사람을 만나고 싶습니다.
